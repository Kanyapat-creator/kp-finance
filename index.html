Format Guide
Sales file (.XLS from the KP accounting system):
The system reads columns: day month year | Total | Goods Value | VAT | Paid | Outstanding

Daily expense file (.xlsx):
The header row contains the word "วันที่" (Date) · must include "รายจ่ายรายวัน" (Daily Expense) and "จำนวนลูกค้า" (Customers) columns · other columns in between are treated as KPI categories
The table header states the month/year, e.g. "...เดือน พฤษภาคม ปี 2569"

Clear All Data?

This will permanently delete all sales, expenses, KPI, and opening balance data from the system. This cannot be undone.
